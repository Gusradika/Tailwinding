## Text Align

```
class='text-{value}'
```

- `left` text-align: left;
- `center` text-align: center;
- `right` text-align: right;
- `justify` text-align: justify;
- `start` text-align: start;
- `end` text-align: end;

## Font Size

```
class ='text-{value}'
```

- `xs`
- `sm`
- `base`
- `lg`
- `xl`
- `2xl`
- `3xl`
- `...`
- `8xl`
- `9xl`

## Text Decoration

```
class ='underline'
class ='overline'
class ='line-through'
class ='no-underline'
```

## Text Transform

```
class='uppercase'
class='lowercase'
class='capitalize'
class='normal-case'
```
