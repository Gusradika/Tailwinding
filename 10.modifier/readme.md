# 📏 Tailwind CSS - Modifier (Pseudo-class)

## Modifier Syntax

```
class='{modifier}:{class}'
```

modifier

- `hover`
- `focus`
- `first`
- `odd`
- `before`
- `after`
- `placeholder`
- `marker`

class

- can be any

### example

```
class='focus:bg-rose-50'
```

Modifier -> focus
Class -> bg-rose-50
