# üìè Tailwind CSS - Flexbox

## Flexbox syntax

```
class='flex'
```

## Flex direction

```
class='flex-{direction}'
```

- `row`
- `row-reverse`
- `col`
- `col-reverse`

## Justify Content

```
class='justify-{value}'
```

- `normal`
- `start`
- `end`
- `center`
- `between`
- `around`
- `evenly`
- `stretch`

## Align items

```
class='items-{value}'
```

- `start`
- `end`
- `center`
- `baseline`
- `stretch`

## Gap

```
class='gap-{value}'
class='gap-[x/y]-value'
```

## Flex Grow

```
class='grow-{-0}'
```

## Flex Wrap

```
class='flex-{value}'
```

- `wrap`
- `wrap-reverse`
- `nowrap`
